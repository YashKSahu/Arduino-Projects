# ADXL345 3-Axis Accelerometer

![p1](adxl345.jpg)

## Hardware Required
- Arduino Uno
- ADXL345 module
- Breadboard
- Jumper Wires
- Arduino IDE

## Connections
![p2](1.png)

## Result (Arduino IDE)

1. Copy the code in a newfile in Arduino IDE
2. Click Compile
3. Plug in the Arduino Uno with the connections completed
4. Click Upload
5. To see the output, open **Tools**->**Serial Monitor**
